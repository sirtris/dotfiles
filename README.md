# My dotfiles

## Requirements
 - git
 - gnu stow
 
 
## Installation
`sudo apt install stow build-essential`  

## Usage
`git clone git@github.com:sirtris/dotfiles.git`  
`cd dotfiles`

then use gnu stow:  
`stow .`
